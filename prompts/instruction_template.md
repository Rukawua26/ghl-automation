# Plantilla rapida

Pega aqui tus notas como salgan naturalmente. El asistente intentara ordenarlas y solo pedira lo faltante.

Ejemplo:

- No usar SMS, solo WhatsApp y correo
- Revisar workflow actual y compararlo con Clientes potenciales
- Eliminar etapas: Cliente cerrado, No interesados
- Cambiar Lead Entrante por este mensaje: ...
- En No contesta 3er llamado enviar correo `emails/no_contesta_3er_llamado.html`
- Para citas usar un campo de fecha y hora y programar recordatorios 24h, 3h y 1h antes
